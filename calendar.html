Calendar

Mobile, UI component

Demos showing calendar features
for Telerik UI for iOS tablet app and Telerik UI for Android app.

This mobile component was created for cross-platform use in iOS, Android and Windows Phone, following their interaction guidelines and including the most common features the app developers were seeking in a calendar.

My process

- Research

- Create the interaction design through sketches and wireframes

- Discuss and iterate with the team

- Create the default UI of the component for iOS, Android and WP, following their visual guidelines

- Design demos, based on the identified real-life scenarios in the research phase, which showcase the UI component in the best way
