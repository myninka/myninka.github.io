Data Form

Mobile, Desktop, UI component

Demos showing data form features
for Telerik UI for Android app.

The task was to design a data form custom component for cross-platform use (iOS, Android, Windows Phone, and Windows 10) which follows the best practices of digital form design in layout and validation. The demos with the component show its usage in a real-life app.

My process

- Research

- Create the interaction design through sketches and wireframes

- Discuss and iterate with the team

- Design demos, based on the identified real-life scenarios in the research phase, which showcase the UI component in the best way
