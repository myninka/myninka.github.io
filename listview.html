List View

Mobile, Desktop, UI component

Demos showing list view features
for Telerik UI for iOS tablet app and Telerik UI for Android app.

The List View (data list) is a custom UI component, created for cross-platform use in iOS, Android and Windows, following their interaction guidelines and including the most common features for a list - different layouts, grouping, filtering, selection, reorder, swipe to refresh, swipe to execute.

My process

- Research

- Create the interaction design through sketches and wireframes

- Discuss and iterate with the team

- Create the default UI of the component for iOS, Android and Windows, following their visual guidelines

- Design demos, based on the identified real-life scenarios in the research phase, which showcase the UI component in the best way
