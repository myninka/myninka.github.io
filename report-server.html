Report Server and Report Designer

Web, Desktop, App

Telerik Report Server web application

Report Server is a web application to store and manage files (reports). It is an industry standard enterprise app - it has interface for editing and managing reports, users and user roles, scheduling. It is designed responsive, for a heavy user input with a lot of specifics for the reporting field.

 

Report Designer is a desktop app for creating and editing reports, it counts on users’ knowledge to work with data sources, analytics and data visualization tools. Report Server and Report Designer compliments each other. Although both of the apps’ users have to be trained professionals to use the apps, the goal was the same as designing for any type of user -  to improve their experience.

Report Server
One thing we have under-thought was the use of popups for user input which are not the most appropriate UI for smaller form factors. However, we saw the need of improvement.
 
Report Server
Custom icon font was made to allow customers to customize their UI and to prepare the interface for smaller form factors and high resolution screens.
Report Designer
Applying the UI patterns Microsoft used in their apps - very dense and option-rich UI. The goal was to make it familiar and easier to execute the complex tasks.
Report Designer
Wizards were chosen as the main UI pattern to allow the users to add complex elements to the report. The goal was to educate and guide users through the interface.
