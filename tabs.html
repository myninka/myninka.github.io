Tabs

Mobile, Desktop, UI component

Demos showing tabs features
for Telerik UI for iOS app and Examples NativeScript.

Tabs component is a custom UI component, created for cross-platform use in iOS, Android and Windows, following their interaction guidelines and including the most common features the app developers were seeking in a tab bar. The user can interact with the component with touch or with mouse (or pen).

My process

- Research

- Create the interaction design through sketches and wireframes

- Discuss and iterate with the team

- Create the default UI of the component for iOS, Android and Windows, following their visual guidelines

- Design demos, based on the identified real-life scenarios in the research phase, which showcase the UI component in the best way
