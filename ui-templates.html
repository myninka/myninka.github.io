UI Templates

Mobile, UI component

The task was to identify and create cross-platform mobile UI patterns which will be most helpful for developers to assemble their app without having a designer around. The UI patterns grew up to be templates - with the desired functionality and basic visual design applied including the transitions, ready to be used with just a little customization. They come together with a mobile UI theme which polishes the UI.

 

The templates target was the NativeScript developers and addresses their need of polished UI and functionality packed in a cross-platform building block, allowing them to focus on the core of their apps - the business logic.

 

The templates consist of:

Views

 

Screens with diferent layouts, scenarios and purpose

that an app could be built from:

 

  1. List views (Data lists): Activity feeds, cards, comments, lists, etc.
  2. Data entry (Forms): Login, sign up, multi-step, check-out, etc.
  3. User profiles: Social platforms user prole screens
  4. Content screens
  5. Product details: Content screens for the ecommerse
  6. Settings/About
  7. Calendar: Full screen with inline events, agenda views
  8. Charts: Dashboards/data
  9. Maps: Integration with Google Maps (other provider)
  10. Messages: Conversation views

Workflows

 

Ready workflows (user flows) from the views to accomplish

the following tasks:

 

  1. Navigation/exploring: Primary/secondary navigation patterns
  2. Search, sort and filter
  3. Login/sign up: Including social login
  4. Onboarding: Login/sign up and walktroughs
  5. Messaging
  6. Check-out: Ecommerce check-out
